# GithubUsers
Показывает информацию о пользователях на гитхабе в виде списка. При тапе на юзера показывает подробную информацию о пользователе. 

Используется:
1) архитектура MVP
2) База данных Room
3) Retrofit
4) Dagger
5) ViewModel
6) ButterKnife

Для тестов:
1) Mockito
2) Robolectric
3) Espresso
4) Junit
